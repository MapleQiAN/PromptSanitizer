# 隐私声明

## 本地优先原则

PromptSanitizer 严格遵循"本地优先"原则：

1. **完全离线运行**: 所有处理都在本地计算机上进行，不会发送任何数据到网络
2. **无数据收集**: 不收集、不存储、不上传任何用户数据
3. **无日志记录**: 不记录原文或完整敏感信息到日志文件
4. **无追踪**: 不包含任何分析或追踪代码

## 数据处理

### 输入数据

- 用户输入的文本完全在本地内存中处理
- 不会保存到磁盘（除非用户主动导出）
- 不会发送到任何服务器

### 输出数据

- 清洗后的文本仅显示在应用界面
- 导出的报告不包含完整敏感内容，只包含掩码预览
- 用户可以随时关闭应用，所有数据从内存中清除

### 文件处理

- 导入的文件仅在读取时加载到内存
- 不会在磁盘上创建临时文件
- 不会上传文件内容

## 安全措施

1. **敏感信息保护**: 
   - 报告中的敏感内容仅显示部分掩码预览
   - 日志中不输出完整敏感字符串

2. **错误处理**:
   - Go 引擎崩溃时，错误信息不包含敏感数据
   - 错误提示仅包含技术信息

3. **代码审计**:
   - 所有代码开源，可自行审查
   - 不包含任何隐藏的网络请求

## 未来功能

如果未来添加 LLM 增强模式（可选功能）：
- 必须用户明确启用
- 会明确告知数据将被发送到外部服务
- 提供详细的隐私政策说明

## 用户权利

用户有权：
- 查看所有源代码
- 自行构建和运行应用
- 修改代码以满足自己的隐私需求
- 不连接任何网络服务

如有隐私相关问题，请通过项目仓库的 Issues 联系。
