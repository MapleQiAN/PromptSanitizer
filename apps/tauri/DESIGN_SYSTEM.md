# 🎀 PromptSanitizer - Kawaii Design System

## 设计理念 / Design Philosophy

这是一个充满活力和可爱感的全新设计系统，完全推翻了原有的工业风格。新设计采用**活力可爱风（Kawaii/Playful Style）**，营造出温暖、友好、充满生命力的用户体验。

## 核心特点 / Core Features

### 1. 🎨 色彩系统 / Color System

#### 浅色主题（默认）
- **主色调**: 粉红色系 (#FF6B9D, #FFA4C4)
- **次要色**: 青蓝色 (#00D9FF, #5FE4FF)
- **强调色**: 明黄色 (#FFD93D)
- **背景**: 温暖的粉白色系 (#FFF4F8, #FFFAFC, #FFE8F0)

#### 深色主题
- **主色调**: 保持粉红色但更加柔和
- **背景**: 梦幻紫黑色系 (#1A1625, #251E35, #342A45)
- **装饰色**: 薰衣草紫、薄荷绿

### 2. 🔤 字体系统 / Typography

- **展示字体**: Fredoka - 圆润可爱，富有个性
- **正文字体**: Quicksand - 现代、友好、易读
- **等宽字体**: JetBrains Mono - 专业而不失时尚

### 3. 🎭 视觉元素 / Visual Elements

#### 形状
- **圆角**: 12px-40px 大圆角设计
- **无尖锐边角**: 所有元素采用柔和的圆润设计

#### 阴影
- 多层次柔和阴影系统
- 使用主色调创建发光效果
- 悬停时阴影动态增强

#### 装饰
- **浮动emoji**: 🌸✨💖⭐🦋 作为背景装饰
- **渐变气泡**: 柔和的径向渐变作为氛围装饰
- **闪烁特效**: ✨ 等emoji的动态旋转动画

### 4. 🎬 动画系统 / Animation System

#### 进场动画
- **弹性进场** (ease-spring): 卡片、侧边栏
- **淡入上浮**: 内容区块
- **缩放淡入**: 面板元素

#### 交互动画
- **悬停抬升**: 按钮、卡片向上浮动
- **缩放反馈**: 复选框、小元素的缩放
- **颜色过渡**: 边框、背景的柔和渐变

#### 装饰动画
- **脉冲发光**: Logo、主按钮
- **浮动**: 气泡、装饰元素
- **闪烁旋转**: emoji装饰

### 5. 🌐 国际化 / Internationalization

支持中英文双语切换：
- 完整的i18n系统
- 语言切换按钮位于顶部栏
- 所有界面文本都支持切换

### 6. 🌗 主题切换 / Theme Toggle

- 流畅的深浅色主题切换
- 默认使用浅色主题
- 所有颜色变量动态响应
- 平滑的过渡动画

## 组件设计 / Component Design

### 侧边栏 (Sidebar)
- 圆润的卡片式设计
- Logo 带有弹跳动画
- 控制按钮采用渐变背景
- 悬停时按钮向右位移

### 顶部栏 (Header)
- 视图切换器采用圆角标签页设计
- 语言和主题切换按钮
- 柔和的阴影效果

### 主内容区 (Main Content)
- 圆角面板卡片
- emoji 装饰在角落
- 悬停时卡片上浮
- 渐变色进度条

### 底部发现列表 (Findings List)
- 圆角表格设计
- 行悬停时缩放效果
- 彩色风险徽章
- 可爱的空状态提示

### 报告视图 (Report View)
- 大型圆角风险评分卡
- 统计卡片悬停动画
- 渐变边框和发光效果
- 居中导出按钮

## 技术实现 / Technical Implementation

### CSS 变量
所有颜色、间距、圆角都使用 CSS 变量定义，便于主题切换和维护。

### 动画曲线
- `ease-bounce`: 弹性效果
- `ease-smooth`: 平滑过渡
- `ease-spring`: 弹簧效果

### 响应式
- 网格布局自适应
- 灵活的 flexbox 设计
- 流式内容区域

## 设计灵感 / Design Inspiration

- **Nintendo Switch** UI - 活泼的色彩和圆润的设计
- **Kawaii Culture** - 日本可爱文化的温暖感
- **Modern Web Apps** - 当代 Web 应用的最佳实践
- **Glassmorphism** - 柔和的背景模糊效果

## 与原设计的对比 / Comparison with Original Design

| 方面 | 原设计 (Industrial) | 新设计 (Kawaii) |
|------|-------------------|----------------|
| 色彩 | 军绿色、橙色 | 粉红、青蓝、明黄 |
| 字体 | IBM Plex (专业) | Fredoka/Quicksand (可爱) |
| 形状 | 斜角切割 | 大圆角 |
| 氛围 | 严肃、技术感 | 温暖、友好、活泼 |
| 动画 | 简单线性 | 弹性、活泼 |
| 装饰 | 扫描线、网格 | emoji、气泡、渐变 |

## 使用指南 / Usage Guide

1. **主题切换**: 点击右上角的太阳/月亮按钮
2. **语言切换**: 点击右上角的地球图标
3. **视图切换**: 使用顶部的分栏/对比/报告标签
4. **交互**: 所有按钮和卡片都有悬停效果，尽情探索！
